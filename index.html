A feature-rich clone of a movie streaming site made with React / Redux

Live page (on Netlify) Source code Original site

This is my biggest and most complex work so far. It's a partial clone of a big commercial site where I checked the original site's look and functionality and implemented the same without copying any of the site's code.

  • Responsive design implemented on each page. Separate navigation/search menu for mobile mode
  • A complex Sort and Filter widget stores all its state data in Redux
  • Fully functional search bar (matches movies where title contains all words from the query)
  • Movie data is hosted on Google Firebase. Each movie page view updates the viewcount in the database
  • Decorative elements on all pages move in response to mouse cursor movement, which is tracked using a custom hook

A modular calculator with drag-and-drop features and complex logic. Made with React / Redux

Live page (on Netlify) Source code

This is my latest work so it's the best example of my code quality.

This project heavily focuses on logic while using barebones styling. Doesn't work on mobile devices as they don't support drag-and-drop natively, but I might look into it later.

  • Drag and drop any module from the left pane to add it to the calculator
  • Rearrange modules in the calculator by dragging them. Display module is always on top and it can't be moved
  • Remove modules from the calculator by double clicking them
  • Click the "Runtime" button to enable the calculator's buttons. It will work even if not all modules are added.

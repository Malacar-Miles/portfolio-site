A feature-rich clone of a movie streaming site made with React

Live page (on Netlify) Source code Original site

This is my biggest and most complex work so far. I'm still adding small features and polish from time to time, but it's mostly complete.

  • Responsive design implemented on each page. Separate navigation/search menu for mobile mode
  • A complex Sort and Filter widget stores all its state data in Redux
  • Fully functional search bar (matches movies where title contains all words from the query)
  • Movie data is hosted on Google Firebase. Each movie page view updates the viewcount in the database
  • Decorative elements on all pages move in response to mouse cursor movement, which is tracked using a custom hook

A code-along project: e-commerce site made with React

Live page (on Netlify) Source code Design and tutorial source

I didn't develop this app myself — it's a code-along project, i.e. I made it by following this online training course. However I do understand each step so I can reproduce the whole project, and I've implemented some parts of the app on my own (wherever the instructor suggested this as a challenge for students).

  • Uses Firebase to store product data and authenticate users
  • Supports Google sign in, and also sign up / sign in with email and password
  • Fully functional Product Cart
  • Uses Redux to manage all state data
  • Uses react-router-dom library to implement navigation
